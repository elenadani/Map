Введение

1. Перечень используемых терминов и библиотек
Интерактивная карта – это электронная карта, работающая в режиме двухстороннего диалогового взаимодействия человека (пользователя) и компьютера, и представляет собой визуальную информационную систему.

Python – это высокоуровневый язык программирования.

PyCharm – это интегрированная среда для разработки приложений на языке Python.

База данных – это упорядоченный набор структурированной информации или данных, которые обычно хранятся в электронном виде в компьютерной системе.

SQLite – это компактная встраиваемая система управления базами данных.

API (Application Programming Interface) – это программный интерфейс приложения, описание способов взаимодействия одной компьютерной программы с другими.

API-ключ – это уникальный код, используемый API для идентификации приложения или пользователя.

HERE maps – это международная технологическая компания, которая занимается разработкой картографического контента, предоставляет данные о местоположении, а также сопутствующие услуги частным лицам и компаниям.

HTML – это стандартизированный язык гипертекстовой разметки документов для просмотра веб-страниц в браузере.

JavaScript – это язык программирования, который используют для разработки интерактивных веб-страниц.

Фреймворк – это готовый набор инструментов, который помогает разработчику быстро создать продукт.

Flask – это фреймворк для создания веб-приложений на языке программирования Python.

Цель и задачи работы

Цель проекта: создать интерактивную карту-энциклопедию технических изобретений 19-20 веков с помощью среды разработки PyCharm.

Задачи проекта:
- собрать информацию из различных источников по данной теме;
- создать базу данных на основе информации об изобретениях;
- рассмотреть программное обеспечение для создания интерактивной карты;
- создать приложение для визуализации карты;
- тестирование приложения;

Методика выполнения работы

Методы работы: теоретический (поиск и изучение информации по теме) и практический (программирование и тестирование приложения).

Ход работы:
1. Поиск информации о технических изобретениях 19-20 веков в интернете;
2. Создание базы данных на основе информации об изобретениях с помощью программы SQLite;
3. Получение API-ключа для выполнения вызова к HERE maps из приложения;
4. Создание HTML файла интерактивной карты с встроенным кодом JavaScript;
5. Создание приложения на языке Python с помощью веб-фреймворка Flask для визуализации карты;
6. Подключение базы данных для рендеринга информации об изобретениях на карту;
7. Запуск приложения и тестирование проекта;

Результаты

В результате работы над проектом с помощью среды разработки PyCharm была создана интерактивная карта-энциклопедия с информацией об изобретениях, созданных в 19-20 века.

Описание завершённого продукта

Интерактивная карта-энциклопедия, на которой отмечены места, где были созданы изобретения в 19-20 веке. Каждое изобретение отмечено уникальной иконкой, при нажатии на которую появляется информация об выбранном изобретении. Карта имеет функцию просмотра изобретений, сделанных только в 19 или только в 20 веке. Также можно посмотреть информацию о проекте.
